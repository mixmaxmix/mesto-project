# Проект: Место

### Описание

Проект от Яндекс.Практикум где были использованы технологии такие как:
  * Grid
  * Flex
  * media запросы
  * hover эффекты
  * Воссоздание popup (пока без js)
  * Методология БЭМ
  * Git
  * GitHub Pages

Верстка проходила с использованием макета в Figma и проверкой через расширение браузера PerfectPixel.
